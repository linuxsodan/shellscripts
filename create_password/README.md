# パスワード生成

ランダムにパスワードを生成します。

### 機能

- オプションで文字種類選択
- パスワード文字数指定 ~まだ

### 使い方

１０文字のパスワード生成

デフォルトでアルファベット大文字、小文字、数字の組合せ
```
./create_password.sh
9Tr7xLtqFZ
```

オプション`-dl`で数字とアルファベット小文字
```
./create_password.sh -dl
vkodq9pj2e
```

オプション`-d`で数字のみ
```
./create_password.sh -d
2445391408
```

オプション
d 数字
l 英子文字
u 英大文字

まだ文字数指定の機能はない
とりあえず文字種類はできた
